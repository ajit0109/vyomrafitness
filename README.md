# vyomrafitness
A full-stack Gym Management System built for portfolio and academic purposes. Features secure admin authentication, member and trainer management, attendance tracking, membership plans, payment records, and CRUD operations with MySQL integration. Responsive, user-friendly, and deployed live on InfinityFree.
